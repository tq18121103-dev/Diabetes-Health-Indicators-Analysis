# Diabetes-Health-Indicators-Analysis
Statistical analysis and predictive modeling of diabetes risk using BRFSS 2015 dataset (GLM, GAM, LASSO).

## Overview

This project analyzes the relationship between diabetes and various health/lifestyle factors using statistical methods and predictive modeling.

The dataset is derived from the **Behavioral Risk Factor Surveillance System (BRFSS) 2015** by the CDC, containing over **250,000 observations** and **22 variables** related to health conditions and demographics.

---

## Objectives

* Explore and describe key health indicators (BMI, Age, Blood Pressure, Cholesterol)
* Analyze relationships between diabetes and risk factors
* Apply statistical inference methods
* Build predictive models for diabetes classification

---

## Methods

### Data Analysis

* Exploratory Data Analysis (EDA)
* Distribution visualization
* Class imbalance analysis

### Statistical Methods

* ANOVA
* Fisher–Pitman permutation test
* Permutation t-test with Holm correction
* Bootstrap estimation

### Modeling

* Logistic Regression (GLM)
* Generalized Additive Model (GAM)
* Logistic LASSO

### Evaluation

* Confusion Matrix
* Odds Ratio interpretation
* Model comparison

---

## Key Findings

* Diabetes is strongly associated with:

  * High blood pressure
  * High cholesterol
  * Age
  * BMI
* Dataset is **highly imbalanced** (~84% non-diabetic)
* Advanced models (GAM, LASSO) improve interpretability and performance
